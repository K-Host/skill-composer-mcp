# writer-skill

AI 驱动的写作辅助技能，支持多种文体和风格。

## 能力：多风格创作

支持多种写作风格。

- 学术写作：严谨规范
- 商务写作：简洁高效
- 创意写作：生动有趣
- 技术写作：清晰准确

## 能力：内容润色

对已有内容进行优化。

- 语法纠错
- 用词优化
- 句式调整
- 逻辑梳理

技术依赖：openai, anthropic

## 模块：大纲生成

根据主题自动生成文章大纲。

- 层级化结构
- 逻辑递进
- 自动补充子主题

## 模块：摘要提取

从长文中提取摘要。

- 支持提取式和生成式
- 可调长度
- 多语言支持

## 模块：SEO写作

面向搜索引擎的内容创作。

- 关键词密度控制
- 标题优化
- 元描述生成
- 内链建议

## 模块：格式化输出

将内容转换为多种格式。

- Markdown
- HTML
- PDF
- Word

技术依赖：fastapi, flask

## 错误处理

LLM 调用失败时使用缓存结果，支持降级输出。

## 缓存策略

生成内容缓存 30 分钟，避免重复调用 LLM。

## 批处理

支持批量生成文章，流水线处理提高效率。
